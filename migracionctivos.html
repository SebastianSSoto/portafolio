Proyecto Migración del Ecosistema

Fase 1: Desarrollo del Pipeline ETL Automatizado

En esta etapa se construyó un pipeline con Python para realizar el proceso ETL (Extracción, Transformación y Carga). Se automatizó la lectura de archivos Excel, aplicando limpieza, clasificación por criterios y filtros específicos para organizar los datos de forma estructurada y eficiente.

Fase 2: Implementación del Backend de Ejecución

Se desarrolló un script backend en Python que se conecta con la interfaz web, permitiendo la ejecución de los distintos procesos ETL mediante solicitudes controladas, de forma segura e intuitiva para el usuario.

Fase 3: Diseño de la Interfaz Interactiva (Frontend)

Se creó una interfaz web limpia y amigable, con botones funcionales que permiten al usuario ejecutar los distintos scripts sin necesidad de conocimientos técnicos. Esta interfaz mejora la experiencia y facilita el uso del sistema.

Fase 4: Visualización y Automatización de Funcionalidades

Se incorporó una capa visual intuitiva que permite acceder rápidamente a las distintas funciones del sistema. Cada botón ejecuta tareas automatizadas como carga a MySQL o procesamiento de datos, mejorando la eficiencia operativa.

Fase 5: Migración Exitosa a Base de Datos Normalizada

Como resultado, se logró ejecutar un proceso completo de toma de un archivo Excel, su transformación con Python y su migración a una base de datos MySQL ya normalizada. Este flujo sirve como modelo para futuras integraciones.

Resultado del Proyecto

Este proyecto permitió consolidar un ecosistema de procesamiento de datos más eficiente, intuitivo y escalable. Se logró automatizar tareas repetitivas, mejorar la organización de la información, facilitar la interacción con los scripts y migrar datos a un entorno estructurado y profesional.
